# Morse Code Interpreter

## Description
This project is a Morse Code Interpreter designed to convert Morse code signals into readable text.

## Objectives
- Detect Morse code inputs
- Interpret dots and dashes
- Convert signals into readable characters
- Improve understanding of embedded systems and signal processing

## Components
- Arduino
- Buzzer
- Potentiometer
- Breadboard
- Jumper Wires

## Features
- Real-time decoding
- Adjustable sensitivity
- Simple and practical system design

## Technologies Used
- Arduino
- Embedded Systems
- Signal Processing
- Circuit Design

## Future Improvements
- LCD display integration
- More accurate signal detection
- Expanded character support
